## Platform setup

**macOS** — Grant accessibility permissions to your terminal (or IDE) in **System Settings → Privacy & Security → Accessibility**.

**Linux** — Ensure AT-SPI2 is running (default on GNOME and most desktop environments). No special permissions needed.

**Windows** — No special permissions needed.

## Example

<Tabs syncKey="lang">
  <TabItem label="Rust">
    ```rust
    use xa11y::*;
    use std::time::Duration;

    fn main() -> Result<()> {
        // Check that we have accessibility permissions
        check_permissions()?;

        // Snapshot the Calculator app's accessibility tree
        let root = app(&AppTarget::ByName("Calculator".to_string()))?;

        // Print the full tree for debugging
        println!("{root}");

        // Find all buttons in the snapshot
        let buttons = root.query("button")?;
        println!("Found {} buttons", buttons.len());

        // Read properties from a specific node
        let equals = root.query("button[name='=']")?;
        let btn = &equals[0];
        println!("role={} name={:?} bounds={:?}", btn.role, btn.name, btn.bounds);

        // Create a locator for actions — it refetches the tree on every call,
        // so it always targets the current UI state
        let seven = locator(
            AppTarget::ByName("Calculator".to_string()),
            "button[name='7']",
        )?;
        seven.press()?;

        // Type into a text field
        let input = locator(
            AppTarget::ByName("Calculator".to_string()),
            "text_field[name='Input']",
        )?;
        input.type_text("42")?;

        // Wait for UI changes before continuing
        let result = locator(
            AppTarget::ByName("Calculator".to_string()),
            "static_text[name='Result']",
        )?;
        result.wait_visible(Duration::from_secs(5))?;
        println!("Result: {:?}", result.value()?);

        // List all running apps
        let all = apps()?;
        for child in all.children() {
            println!("{} (pid: {:?})", child.name.as_deref().unwrap_or("?"), child.pid);
        }

        Ok(())
    }
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import xa11y

    # Check that we have accessibility permissions
    xa11y.check_permissions()

    # Snapshot the Calculator app's accessibility tree
    root = xa11y.app("Calculator")

    # Print the full tree for debugging
    print(root)

    # Find all buttons in the snapshot
    buttons = root.query("button")
    print(f"Found {len(buttons)} buttons")

    # Read properties from a specific node
    btn = root.query("button[name='=']")[0]
    print(f"role={btn.role} name={btn.name} bounds={btn.bounds}")

    # Create a locator for actions — it refetches the tree on every call,
    # so it always targets the current UI state
    seven = xa11y.locator("Calculator", selector="button[name='7']")
    seven.press()

    # Type into a text field
    input = xa11y.locator("Calculator", selector="text_field[name='Input']")
    input.type_text("42")

    # Wait for UI changes before continuing
    result = xa11y.locator("Calculator", selector="static_text[name='Result']")
    result.wait_visible(timeout=5.0)
    print(f"Result: {result.value()}")

    # List all running apps
    all_apps = xa11y.apps()
    for child in all_apps.children:
        print(f"{child.name} (pid: {child.pid})")
    ```

## Selector syntax

| Pattern                        | Meaning                          |
| ------------------------------ | -------------------------------- |
| `button`                       | Elements with role `Button`      |
| `button[name='OK']`            | Button named exactly "OK"        |
| `textfield[name^='Search']`    | Text field whose name starts with "Search" |
| `textfield[name*='email']`     | Text field whose name contains "email" |
| `group > button`               | Buttons that are direct children of a group |
| `window button[name='OK']`     | Button named "OK" anywhere inside a window |
| `button:nth(2)`                | The 2nd button match             |
