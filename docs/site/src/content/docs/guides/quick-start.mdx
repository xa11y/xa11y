## Platform setup

**macOS** — Grant accessibility permissions to your terminal (or IDE) in **System Settings → Privacy & Security → Accessibility**.

**Linux** — Ensure AT-SPI2 is running (default on GNOME and most desktop environments). No special permissions needed.

**Windows** — No special permissions needed.

## Example

<Tabs syncKey="lang">
  <TabItem label="Rust">
    ```rust
    use xa11y::*;
    use std::time::Duration;

    fn main() -> Result<()> {
        // Check that we have accessibility permissions
        check_permissions()?;

        // Connect to the Calculator app
        let calc = App::from_name(provider()?, "Calculator")?;

        // Snapshot the tree for debugging
        let root = calc.elements()?;
        println!("{root}");

        // Find all buttons via locator
        let buttons = calc.locator("button").elements()?;
        println!("Found {} buttons", buttons.len());

        // Read properties from a specific element
        let btn = calc.locator("button[name='=']").element()?;
        println!("role={} name={:?} bounds={:?}", btn.role, btn.name, btn.bounds);

        // Locators refetch the tree on every call,
        // so they always target the current UI state
        calc.locator("button[name='7']").press()?;

        // Type into a text field
        calc.locator("text_field[name='Input']").type_text("42")?;

        // Wait for UI changes before continuing
        let result = calc.locator("static_text[name='Result']");
        result.wait_visible(Duration::from_secs(5))?;
        println!("Result: {:?}", result.element()?.value);

        // List all running apps
        for app in App::all(provider()?)? {
            println!("{} (pid: {})", app.name(), app.pid());
        }

        Ok(())
    }
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import xa11y

    # Check that we have accessibility permissions
    xa11y.check_permissions()

    # Connect to the Calculator app
    calc = xa11y.app("Calculator")

    # Snapshot the tree for debugging
    root = calc.elements()
    print(root)

    # Find all buttons via locator
    buttons = calc.locator("button").elements()
    print(f"Found {len(buttons)} buttons")

    # Read properties from a specific element
    btn = calc.locator("button[name='=']").element()
    print(f"role={btn.role} name={btn.name} bounds={btn.bounds}")

    # Locators refetch the tree on every call,
    # so they always target the current UI state
    calc.locator("button[name='7']").press()

    # Type into a text field
    calc.locator("text_field[name='Input']").type_text("42")

    # Wait for UI changes before continuing
    result = calc.locator("static_text[name='Result']")
    result.wait_visible(timeout=5.0)
    print(f"Result: {result.element().value}")

    # List all running apps
    for app in xa11y.apps():
        print(f"{app.name} (pid: {app.pid})")
    ```

## Selector syntax

| Pattern                        | Meaning                          |
| ------------------------------ | -------------------------------- |
| `button`                       | Elements with role `Button`      |
| `button[name='OK']`            | Button named exactly "OK"        |
| `textfield[name^='Search']`    | Text field whose name starts with "Search" |
| `textfield[name*='email']`     | Text field whose name contains "email" |
| `group > button`               | Buttons that are direct children of a group |
| `window button[name='OK']`     | Button named "OK" anywhere inside a window |
| `button:nth(2)`                | The 2nd button match             |
