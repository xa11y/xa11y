## Key Types

### Provider

The main entry point. Create one with `xa11y::create_provider()`.

```rust
use xa11y::*;

let provider = create_provider()?;

// Check permissions
let status = provider.check_permissions()?;

// List running applications
let apps = provider.list_apps()?;

// Get an app's accessibility tree
let tree = provider.get_app_tree(
    &AppTarget::ByName("Safari".to_string()),
    &QueryOptions::default(),
)?;
```

### Tree

A snapshot of an application's accessibility tree.

```rust
// Query elements with CSS-like selectors
let buttons = tree.query("button")?;
let submit = tree.query_one("button[name='Submit']")?;

// Access the root node
let root = tree.root();

// Create a locator for resilient interaction
let loc = tree.locator("button[name='OK']");
```

### Node

A single element in the tree.

```rust
let node: &Node = tree.root();

println!("Role: {:?}", node.role);
println!("Name: {:?}", node.name);
println!("Description: {:?}", node.description);
println!("Value: {:?}", node.value);
println!("Bounds: {:?}", node.bounding_rect);
println!("States: {:?}", node.states);
println!("Children: {}", node.children.len());
```

### Locator

A resilient element reference that re-queries on each interaction.

```rust
let loc = tree.locator("textfield[name='Email']");

loc.press(&*provider)?;
loc.set_value(&*provider, "user@example.com")?;
loc.focus(&*provider)?;
```

### Error

All operations return `xa11y::Result<T>`, which uses the `xa11y::Error` enum:

- `PermissionDenied` — accessibility access not granted
- `AppNotFound` — target application not running
- `SelectorNotMatched` — no element matched the selector
- `ActionNotSupported` — action not available on this element
- `InvalidSelector` — malformed selector syntax
- `Platform` — OS-level error
- `Timeout` — operation timed out
